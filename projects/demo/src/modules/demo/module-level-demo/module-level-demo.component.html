Module level

I. Description

If MessageBusModule is registered in an module. An single instance of MessageBusService is used for ALL module components.
This can be a single source of truth to exchange messages between components belong to the same module
Every component which broadcasts to the same channel & event can communicate to each other.
Within the module, event component is generated by a router-outlet, it can receive the messages too
